# Create a Warehouse API client

Create a Warehouse API client

## Usage

``` r
new_warehouse_client(config = WarehouseAPIConfig())
```

## Arguments

- config:

  Configuration list for API endpoints and settings. Defaults to
  [`WarehouseAPIConfig()`](https://csiontario.github.io/csiapps/reference/WarehouseAPIConfig.md).

## Value

A list of functions for interacting with the Warehouse API
